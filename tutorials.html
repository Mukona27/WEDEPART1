HTML Fundamentals

1. What is HTML?

HTML (HyperText Markup Language) is the structure of the web. It defines headings, text, images, links, and layouts.

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello World!</h1>
  </body>
</html>

2. Headings & Paragraphs

Headings are from <h1> (largest) to <h6> (smallest). Paragraphs use the <p> tag.

<h1>Main Heading</h1>
<p>This is a paragraph.</p>

Python Basics

1. Hello World

Python is a beginner-friendly language. Your first program prints a message:

print("Hello, World!")

2. Variables

Variables store data values.

name = "Alice"
age = 20
print(name, age)

3. If Statements

x = 10
if x > 5:
    print("x is greater than 5")

JavaScript Basics

1. Hello World

console.log("Hello, World!");

2. Variables

Use let or const.

let name = "Alice";
const age = 20;
console.log(name, age);

3. Functions

function greet(name) {
  return "Hello " + name;
}
console.log(greet("Alice"));
